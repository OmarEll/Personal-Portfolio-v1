RevueOnline

  • Application de gestion des revue en ligne réalisée en Java en utilisant JavaFX
  • Implementation d’une base de données en utilisant JDBC et la méthode DAO et DAOFactory (Choix entre liste mémoire et MySQL)
Objectif du projet

Développer un logiciel en suivant les préceptes de l'agilité  et mettant en œuvre un ensemble de bonnes pratiques

Compétences visées

rogrammation événementielle, JDBC, Design Pattern DAO, architecture MVC, tests unitaires

Langages utilisés:
  • Java
  • JavaFX
  • JUnit
  • mySQL
  • git
  • CSS
